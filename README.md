# 🏥 Hospital Management System

A full-stack web application built using Java technologies to manage hospital operations efficiently with role-based access.

## 🚀 Features
- 🔐 Role-based login (Admin, Doctor, Patient)
- 👨‍⚕️ Admin Dashboard:
  - Manage Doctors (Add/Edit/Delete)
  - Manage Specializations
  - View Patients & Appointments
- 🩺 Doctor Dashboard:
  - View assigned appointments
  - Add patient comments/diagnosis
  - Update profile
- 👤 Patient Module:
  - Register & Login
  - Book appointments
  - View appointment history
  - Change password

## 🛠 Tech Stack
- **Backend:** Java, JSP, Servlets, JDBC  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** MySQL  
- **Server:** Apache Tomcat  
- **Version Control:** Git & GitHub  

## 🧱 Architecture
- MVC (Model-View-Controller)
- DAO Pattern for database operations

## 📊 Database
- Normalized schema with tables:
  - Users
  - Doctors
  - Appointments
  - Specializations

## ⚙️ Setup Instructions
1. Clone the repository  
2. Import into Eclipse as Maven Project  
3. Configure MySQL database  
4. Run on Apache Tomcat Server
